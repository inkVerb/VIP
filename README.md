## Tips

*Ctrl+A* select All

*Ctrl+X* Cut

*Ctrl+Z* Undo

*Ctrl+Shift+Z / Ctrl+Y* Redo

*Ctrl+T* Open a new tab

*Ctrl+W* Close a tab

*Ctrl+LEFT_CLICK* Open link in a new tab

## In Terminal

*Ctrl+Shift+V* Paste

*Ctrl+Shift+T* Open a new tab
